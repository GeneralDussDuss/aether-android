ESSENTIA.JS WASM ANALYSIS
Float-precision BPM from RhythmExtractor2013 — not rounded integers. Key detection, danceability scoring, onset rate analysis. 60-second segment optimization. Background queue pre-analysis so every track is ready before it's needed.
BEAT-PHASE ALIGNMENT
Beats land on the grid — not near it. Modular arithmetic aligns the incoming track's phase to the outgoing beat grid. Beat-locked drift correction runs in every overlapping transition — blend, bass swap, double drop, loop tease, and filter sweep.
CAMELOT KEY MATCHING
Harmonic mixing by the book. Full Camelot wheel implementation — compatible keys scored and prioritized. No clashing transitions. Every mix sounds intentional.
SET JOURNEY PLANNING
Five-phase set arc: warmup → building → peak → cooldown → finale. Tracks momentum across mixes, adapts energy targets per phase. Learns from your behavior — skips = negative feedback, full plays = positive signal.
9 TRANSITION PATTERNS
Blend, cut, bass swap, echo out, loop tease, double drop, backspin, filter sweep, power down. AI selects based on spectral compatibility, vocal clash detection, set journey phase, and pattern variety.
SMART TRACK SELECTION
Scored by BPM affinity (half/double-time aware), Camelot key compatibility, spectral compatibility, energy trajectory, danceability continuity, artist variety, duration filtering, and user behavior learning. Not random — curated by algorithm.
PROGRESSIVE EQ HANDOFF
DJ-standard EQ at 200Hz/1.2kHz/2.5kHz. Highs fade first, then bass swaps on the beat, then mids restore. Master limiter prevents clipping during double drops. Drift correction across all overlapping transitions.
SPECTRAL INTELLIGENCE
Real DFT-based 16-band spectral profiling with Hann windowing. Compatibility scoring detects frequency clashes before overlap. Vocal detection prevents vocal-over-vocal trainwrecks. Novelty-based structure mapping identifies intros, drops, breakdowns, and outros.
FULL KEYBOARD CONTROL
D to toggle DJ mode. 9 transition hotkeys. Level meters, BPM sync with spectral compatibility indicator, set journey phase display, and transition progress — all live.
